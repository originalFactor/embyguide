# 安装

Emby 官方客户端：[点击这里](https://emby.media/download.html) → "Emby Apps & Devices" 自行选择自己系统对应的安装包（有增值服务）

以下提供一些免费无限制使用的方式

**注意：鉴于作者没有Apple系设备所以不确定Apple系统软件是否还是免费的**

**如果不清楚怎么下载/安装软件请查看使用部分**

*2025/12/7: 替换了 EmbyNoisyX*

## Web 端

Emby 的 HTML5 客户端和服务器是绑定的，一般在服务器的 8096 端口提供服务，不过您也可以尝试 [官方提供的版本](https://app.emby.media)

## 桌面端

- Windows: [Hills Lite](https://apps.microsoft.com/detail/9nxnzfrllwzx) （精简版Windows[点这里](./offline-appx.md)）
- Linux: [Tsukimi](https://github.com/tsukinaha/tsukimi)
- MacOS: [VidHub](https://apps.apple.com/us/app/vidhub-video-library-player/id1659622164?platform=mac) （*不提供教程，查看[官网](https://zh.vidhub.okaapps.com/)*；*可能的内购付费*）

以上程序均适用于 X86 架构 64 位 CPU。

## 移动端

- Android: [Ghosten Player](https://github.com/GhostenEditor/Ghosten-Player/releases/)
- iOS: [Filebar](https://apps.apple.com/us/app/fileball/id1558391784?uo=4) （*可能的内购付费*）

以上程序均适用于 ARM 架构 64 位 CPU（苹果 Axx 系列也是）

## TV 端

- Android TV: [Ghosten Player](https://github.com/GhostenEditor/Ghosten-Player/releases/)
- tvOS: [Fileball](https://apps.apple.com/us/app/filebar-media-player/id1558391784?platform=tv) （*不提供教程*；*可能的内购付费*）

以上程序均适用于 ARM 架构 64 位 CPU
