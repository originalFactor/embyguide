# Android TV 使用

此处以 Ghosten Player 为例，[下载链接](https://github.com/GhostenEditor/Ghosten-Player/releases/)

![image-20251212004109120](androidtv.assets/image-20251212004109120.png)

一般选第二个即可，老设备选第四个，安装自行查询自己电视型号。

![image-20251212004144025](androidtv.assets/image-20251212004144025.png)

进入右上角齿轮

![image-20251212004220576](androidtv.assets/image-20251212004220576.png)

进入远程服务器设置

![image-20251212004257033](androidtv.assets/image-20251212004257033.png)

加号

![image-20251212004319945](androidtv.assets/image-20251212004319945.png)

输入完整链接、用户名、密码，User Agent留空，点确认。

![image-20251212004514090](androidtv.assets/image-20251212004514090.png)

回到这个界面，点击新出现的服务器，点√。

![image-20251212004603297](androidtv.assets/image-20251212004603297.png)

回到主页，切换一下标签页再切回来，就可以使用了。
